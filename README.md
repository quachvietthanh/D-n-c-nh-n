# Advanced ATM Simulator

## 📌 Giới thiệu dự án
Dự án mô phỏng hệ thống quản lý tài khoản và giao dịch tại máy ATM. Đây là dự án cá nhân được xây dựng bằng **Java Core thuần**, chạy trên môi trường **Console**, nhằm rèn luyện tư duy thuật toán thông qua Flowchart và áp dụng các tiêu chí kỹ thuật nâng cao của CodeGym.

## 🛠️ Công nghệ ứng dụng
* Ngôn ngữ: Java Core (Java SE)
* Kiến trúc: Phân tầng trừu tượng (3-Tier Architecture)
* Kỹ thuật áp dụng: Generic, Custom Exception, Data Validation (Regex), Threading
* Design Pattern: Singleton Pattern, Factory Pattern
* Phương pháp phát triển: TDD (Test-Driven Development) với JUnit 5
